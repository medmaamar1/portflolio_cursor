# AI Engineer Portfolio Website — Built by Mohamed Maamar

**Designed and developed by Mohamed Maamar (Med).** A dynamic, interactive frontend portfolio showcasing advanced AI and Cybersecurity projects with premium animations and cutting-edge web performance.

---

## Tech Stack
- **Framework:** React 19 + Vite 8
- **Animations:** GSAP (GreenSock Animation Platform) for timeline-based scroll animations
- **Linting:** Oxlint (Rust-based ultra-fast linting)
- **Build Tool:** Vite (ESBuild)

---

## Architecture & Key Features
- **GSAP Integration:** Complex scroll-triggered animations maintaining high frame rates.
- **Vite + React 19:** Lightning-fast HMR and newest React features.
- **Oxc Toolchain:** Rust-based type-aware code quality enforcement.
