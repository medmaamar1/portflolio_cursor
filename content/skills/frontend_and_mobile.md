# Frontend & Mobile Development — Mohamed Maamar

## Technologies
- Next.js 15 (App Router)
- React 19
- TypeScript
- Capacitor (`@capacitor/core`)
- TanStack Query (React Query)
- Streamlit & Gradio

## Experience
I build high-performance, responsive, and dynamic user interfaces that bridge web and native mobile environments.

Using **Next.js 15 (App Router)** and **React 19**, I develop scalable e-commerce and dashboard applications. I rely heavily on **TypeScript** for end-to-end type safety and utilize **TanStack Query** for aggressive client-side caching, data fetching, and stale-time management, drastically improving perceived performance.

To maximize codebase efficiency, I utilize **Capacitor** to compile a single, unified web codebase into native iOS and Android application packages. This allows me to seamlessly interface with native device hardware (such as camera overlay access) while maintaining the agility of web development.

For internal tools and data science applications, I rapidly prototype and deploy interactive, real-time visual dashboards using **Streamlit** and serve LLM-driven UI tools via **Gradio**.
