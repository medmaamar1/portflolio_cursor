## Experience
I design and deploy State-of-the-Art (SOTA) Retrieval-Augmented Generation architectures that ensure LLM applications are factually grounded, hallucination-free, and highly performant over massive datasets.

Rather than relying on basic vector lookups, I implement **Advanced RAG** pipelines. At the database layer, I utilize **`pgvector`** within PostgreSQL, configuring **HNSW (Hierarchical Navigable Small World)** indexes to achieve sub-millisecond Approximate Nearest Neighbor (ANN) search at scale. 

To maximize retrieval precision, I architect **Hybrid Search** algorithms that combine sparse keyword matching (BM25) with dense vector embeddings, merging the results using Reciprocal Rank Fusion (RRF). I further optimize the pipeline by implementing **Cross-Encoder Re-Ranking**, computing attention across the query and the initial retrieved documents to drastically improve precision@k before feeding context to the LLM.

On the ingestion side, I engineer robust data pipelines using advanced semantic chunking and sliding window overlaps. This ensures that long-form contextual meaning is mathematically preserved during the embedding process, eliminating context-fragmentation. All factual statements generated by my agents are strictly anchored to these retrieved contexts, establishing absolute deterministic traceability in AI applications.

To guarantee production-readiness, I implement rigorous automated evaluation of the entire pipeline using **RAGAS** and **LLM-as-a-Judge** paradigms. I systematically compute **Context Precision** and **Context Recall** to benchmark the retrieval architecture, while measuring **Faithfulness** and **Answer Relevance** to quantify the generation layer, strictly catching hallucinations and performance regressions before deployment.
